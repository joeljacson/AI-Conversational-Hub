## Packages
framer-motion | For complex layout transitions and message animations
react-markdown | To render rich text in chat responses if needed
clsx | For conditional class names
tailwind-merge | For merging tailwind classes safely

## Notes
- The app is a single-page chat interface.
- Using 'Outfit' for headings and 'DM Sans' for body text.
- Theme is "Dark Navy" with "Cyan" accents.
- Metadata in messages drives UI elements like Chips and Recipe Steps.
